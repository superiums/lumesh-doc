## 样式

| 设置样式 | 恢复 |
|----------|------|
|   | RESET |
| NORMAL | RESET_NORMAL |
| BOLD | RESET_BOLD |
| DIM | RESET_DIM |
| ITALIC | RESET_ITALIC |
| UNDERLINE | RESET_UNDERLINE |
| BLINK | RESET_BLINK |
| REVERSE | RESET_REVERSE |
| HIDDEN | RESET_HIDDEN |
| STRIKE | RESET_STRIKE |


用法：
```bash
STYLE.DIM + 'lume'
# 效果等同于
string.dim('lume')
```
