## 占位符 `_`

> Lumesh中独有的占位符，具有以下几种用途：

- 在单个单词的命令后，作为空白占位符

```bash
ls _        # 表示这是一条命令

let ls = 5
ls          # 这里会直接打印该变量     
```

- 在区间或切片中，作为开区间

```bash

_..10      # 包含小于10的所有整数

1.._       # 包含大于等于1的所有整数

a[_..10]   # 包含a中所有索引小于10的元素

```

- 在管道右边，表示管道数据的目标位置

```bash
2 | print 1 _ 3     # 管道数据放到第二个参数位置

```
