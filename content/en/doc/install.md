1. download

visit [https://codeberg.org/santo/lumesh/releases](https://codeberg.org/santo/lumesh/releases) to download bin file for your os。

macos: visit [https://github.com/superiums/lumesh/releases](https://github.com/superiums/lumesh/releases) to download。

2. install

copy the bin file to system path, eg：
- windows:
`C:\windows\lume`

- linux:
`/usr/bin/lume`

- macos:
`/usr/bin/lume`

3. permit execution privileage（only linux and macos needed)
`sudo chmod +x /usr/bin/lume`

4. if you want to set as login shell（only linux and macos supported)
run `set_as_login_shell()` in lume.

## update
run `update()` in lume

## help
run `help` in lume to see help of functions；

run `doc` in lume to see help info online.
