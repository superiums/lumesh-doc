## STYLE

| style | reset |
|----------|------|
|   | RESET |
| NORMAL | RESET_NORMAL |
| BOLD | RESET_BOLD |
| DIM | RESET_DIM |
| ITALIC | RESET_ITALIC |
| UNDERLINE | RESET_UNDERLINE |
| BLINK | RESET_BLINK |
| REVERSE | RESET_REVERSE |
| HIDDEN | RESET_HIDDEN |
| STRIKE | RESET_STRIKE |


usage：
```bash
STYLE.DIM + 'lume'
# same as
string.dim('lume')
```
